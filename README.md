# gastown-calculator

A web-based calculator built with TypeScript and Vite.

## Development

```bash
npm install
npm run dev       # Start dev server
npm run build     # Production build
npm run test      # Run tests
```

## Keyboard shortcuts

- `0-9` — digits
- `+-*/` — operators
- `.` — decimal point
- `Enter` / `=` — evaluate
- `Escape` / `Backspace` — clear
